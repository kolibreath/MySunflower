# 使用
本项目和Sunflower一样都使用了[Unsplash api](https://unsplash.com/documentation),为了隐私安全，项目隐去了存有ACCESS_KEY，需要使用项目的话可以
自行创建一个Unsplash 账号获取ACESS_KEY。然后将此行加入gradle.properties的最后一行
````
unsplash_access_key = <your access key>
````
